1. Where should you use constants, and where should you use variables? Can they all be constants?

    2. Konstanter bør brukes for verdier som ikke skal endres under kjøringen av programmet. Dette er altså faste verdier. Variabler brukes før verdier som kan endres under kjøring av programmet, som resultat beregninger osv. Nei ikke alt kan være konstanter fordi variabler må endres i noen visse situasjoner.

  2. How do parentheses affect the arithmetic outcome of an expression?

    3. Parenteser endrer rekkefølgen i beregningen. Det som kommer først i utregningen er det som er innenfor parenteser. Så hvordan du plasserer og bruker parentes vil endre og gi forskjellig svar.

  3. How do we limit floating-point numbers to a given number of decimals?

    4. For å minske desimaler i kan man bruke «round()» eller «toFixed()».

  4. How can we compare two floating-point values to see if they are equal (is 12.199999 the same as 12.2)?

    5. Flyttall kan ha små avrundingsfeil. Så i stedet for å sjekke om de er like, sjekker man ofte forskjellen mellom dem er mindre enn terskelverdi.

  5. What do comparison operators return, and how can we check if they are FALSE or TRUE?

    6. Sammenligningsoperatorer viser en boolsk verdi som enten er sant eller usant (true or false), man bruker dette for å sjekke om noe er sant eller usant.

  6. How does the Math.random() function work, and what can it be used for?

    7. Math.random() brukes når du skal generere ett tall mellom 0 og 1. Det kan brukes for å lage tilfeldige tall, mellom ett ønsket spekter av tall. Det er en nyttig ting for å kunne programmere spill eller for å kunne trekke tilfeldige valg.

  7. What is modulus (%), and when do we need this operator in mathematical operations?

    8. Modulus er en type operatør som gir muligheten til å returnere tall når det deles på et annet.
